Sharing some coding conventions is a key point to make it possible for a team to efficiently collaborate. This rule makes it mandatory to place open curly braces at the end of lines of code.

Noncompliant Code Example

if (condition)
{
  doSomething();
}

Compliant Solution

if (condition) {
  doSomething();
}

Exceptions

Object literals not appearing as first argument can start on their own line.

functionWithObject(parameter1,
   {                                                 //Compliant
        g: "someValue"
   }
);
